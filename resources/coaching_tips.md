# Coaching Tips and Resources

## General Coaching Principles

1. **Safety First**
   - Always check equipment before use
   - Ensure proper spotting
   - Teach proper progressions
   - Know each athlete's current skill level

2. **Positive Reinforcement**
   - Encourage effort and improvement
   - Celebrate small victories
   - Provide constructive feedback
   - Build confidence

3. **Clear Communication**
   - Use consistent terminology
   - Demonstrate skills when possible
   - Break down complex skills
   - Check for understanding

4. **Progressive Training**
   - Master basics before advancing
   - Use appropriate progressions
   - Don't rush skill development
   - Build strength and flexibility
