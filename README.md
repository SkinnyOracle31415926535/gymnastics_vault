# Gymnastics Vault

Welcome to your gymnastics training vault! This repository helps you organize and track all aspects of your gymnastics program.

## Directory Structure

- **team_info/** - Team rosters, athlete profiles, and contact information
- **training/** - Training schedules, lesson plans, and workout routines
- **competitions/** - Competition results, meet schedules, and performance tracking
- **progress_tracking/** - Individual athlete progress notes and evaluations
- **resources/** - Code of points, coaching resources, and reference materials

## Getting Started

1. Fill in your team information in the `team_info/` directory
2. Create training plans in the `training/` directory
3. Track competitions and results in the `competitions/` directory
4. Monitor athlete progress in the `progress_tracking/` directory
5. Add useful resources and references in the `resources/` directory

## Tips

- Keep files organized in their respective directories
- Use clear, descriptive file names with dates (e.g., `2026-02-training-plan.md`)
- Review and update progress notes regularly
- Back up important files frequently
